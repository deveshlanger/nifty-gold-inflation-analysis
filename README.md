# nifty-gold-inflation-analysis
Analysing 15 years of NIFTY 50 vs Gold returns adjusted for inflation — using Python, Pandas and Matplotlib
# Inflation-Adjusted Performance of Indian Equities and Gold

A data analysis project comparing the real (inflation-adjusted) returns 
of NIFTY 50 and gold over 15 years — because most investment discussions 
in India still focus on nominal returns, which can be deeply misleading.

## Why I Built This

Growing up, CNBC and ET Now were always on at home. My dad, like most 
Indian parents, had strong opinions about gold. But as I started learning 
more about markets, I kept wondering — does gold actually hold up when you 
adjust for inflation? And how does it compare to just investing in NIFTY?

Traditional Indian investing has always leaned heavily on gold. A lot of 
people are still hesitant about equities. I wanted to see what the data 
actually says.

## What This Project Does

- Compares nominal vs real (inflation-adjusted) returns of NIFTY 50 and gold
- Analyses performance across three inflation regimes — low, moderate, and high
- Computes Sharpe Ratio to measure risk-adjusted returns for both assets
- Simulates long-term wealth creation from ₹100 invested in each asset

## Data Sources

| Dataset | Source |
|---|---|
| NIFTY 50 Daily Prices | Yahoo Finance |
| Gold Prices | International gold price data |
| India CPI Inflation | World Bank |

## Key Findings

- NIFTY delivers higher average real returns but with significantly more volatility
- Gold outperforms NIFTY during high inflation periods (CPI > 6%)
- A blended 60/40 portfolio reduces risk without proportionally sacrificing returns
- Nominal returns overstate real wealth creation by more than half in several years

## Tools Used

- Python, Pandas, Matplotlib, Seaborn, NumPy
- Jupyter Notebook
